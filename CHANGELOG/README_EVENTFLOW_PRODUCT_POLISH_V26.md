# EventFlow Product Polish V26

Patch przebudowuje widok **Magazyn → Magazyn wewnętrzny**.

## Najważniejsze zmiany

- kategorie główne są pokazane jako krótsza lista chipów,
- podkategorie są rozwijane dopiero po kliknięciu kategorii głównej,
- lista modeli jest czytelniejsza i bliższa widokowi NEW,
- dodano miniaturę zdjęcia modelu w tabeli,
- dodano szybkie akcje: podgląd, edycja, dostępność, szczegóły,
- filtry po kategoriach uwzględniają podkategorie,
- statystyki magazynu liczą się dla aktywnie wybranego filtra.

Kod kreskowy dalej należy do egzemplarza, nie do modelu.
